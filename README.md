# Project Start
- Backend Laravel
  - get current IP

  - ```
    ifconfig
    ```
  - ```
    php artisan serve --host IP
    ```
  - change IP in REST component (axios GET)


- GraphQL Backend
  - ```
    docker-compose up -d
    ```
  - http://localhost:5000/graphiql

  - pgAdmin: http://localhost:5050/browser/
